# 个人.NET Core练习示例仓库

欢迎下载[我的.Net core示例](https://github.com/TanWucheng/MyDotNetRepo.git)仓库

**克隆**

```bash
git clone https://github.com/TanWucheng/MyDotNetRepo.git
```

**Restore**

```bash
dotnet restore
```

**Build\&Run**

```bash
dotnet build && dotnet run
```
